== Description ==
Add option to pay using term credit in your WooCommerce store.

This plugin provides term credit payment facility to wholesale clients in WooCommerce store. Customer will be provided a payment gateway where they will be using term credit to pay for their purchases, and only pay the owed amount when credit term is due.

== Installation ==
1. Unzip and Upload Folder to the /wp-content/plugins/ directory.
2. Activate through WordPress plugin dashboard page.
3. Visit Settings > WC Credit Term Payment Gateway > Configure plugin settings.
4. Save Changes.
